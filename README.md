# Spotify Web Player Clone

A responsive Spotify web player built using HTML5, CSS3, and vanilla JavaScript. This client-side application runs entirely in the browser with no backend or database required.

## Working Features
* **Sidebar Song List:** Browse and select songs from a sidebar navigation panel.
* **Interactive Seekbar:** Real-time progress bar that syncs with track playback and allows scrubbing.
* **Responsive Layout:** Adaptive UI built with CSS Grid and Flexbox for desktop and mobile screens.

## Under Development
* **Playlists:** Custom playlist switching and management.
* **Volume Controls:** Slider controls for volume adjustment and mute toggling.

## Tech Stack
* **HTML5:** Semantic layout and audio elements.
* **CSS3:** Flexbox, Grid, and responsive styling.
* **JavaScript:** Audio state management and DOM manipulation.
