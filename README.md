# Spotify Web Player Clone

A responsive, client-side Spotify web player clone built from scratch using HTML5, CSS3, and vanilla JavaScript. This project is currently in active development and operates entirely in the browser with no backend or database required.

## Features
* **Sidebar Navigation:** Dedicated panel displaying the list of songs for easy browsing and selection.
* **Working Seekbar:** Real-time progress bar that syncs with audio playback and allows seamless track scrubbing.
* **Responsive Design:** Fluid layout using CSS Grid and Flexbox that adapts smoothly across desktop and mobile screens.

## Under Development 🚧
* **Playlists:** Full playlist switching and management functionality.
* **Volume Controls:** Interactive slider for audio level adjustment and mute toggling.

## Tech Stack
* **HTML5:** Semantic structure and native audio elements.
* **CSS3:** Custom properties, Flexbox, Grid, and responsive media queries.
* **Vanilla JavaScript:** DOM manipulation, event handling, and audio state management.
